# 📋 План доработок сайта meritel43.ru

**Дата составления:** 11 марта 2026  
**Текущее состояние:** Сайт запущен локально на PHP 5.6 + MySQL 5.7  
**CMS:** Самописная, версия 3.1.5 (от 2013 года)

## 🔴 Критические проблемы

### Безопасность
| Проблема | Решение |
|----------|---------|
| Пароль БД в коде | Вынести в `.env` |
| PHP 5.6 устарел | Миграция на PHP 8.x |
| Функции `mysql_*` | Замена на PDO/mysqli |
| Нет CSRF-защиты | Добавить токены |
| Слабые пароли | Хеширование bcrypt |
