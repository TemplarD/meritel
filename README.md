# 🛠️ meritel43.ru — Сайт мерительного инструмента

Самописная CMS для интернет-магазина мерительного инструмента.

## ⚠️ Важно

Этот репозиторий содержит **шаблон проекта**. Для работы необходимо:

1. Скопировать `.env.example` в `.env`
2. Настроить параметры подключения к БД
3. Импортировать структуру базы данных

## 🚀 Быстрый старт (Docker)

```bash
# Клонировать репозиторий
git clone https://github.com/TemplarD/meritel.git
cd meritel

# Скопировать конфиг
cp .env.example .env

# Настроить .env (DB_USER, DB_PASSWORD, etc.)

# Запустить
docker-compose up -d

# Импортировать БД (если есть дамп)
docker exec -i meritel43_db mysql -u root -prootpassword viz620 < database/structure.sql
```

Сайт будет доступен по адресу: **http://localhost:8080**

## 🔧 Настройка

### База данных

Отредактируйте `.env`:

```env
DB_HOST=localhost
DB_NAME=viz620
DB_USER=meritel
DB_PASSWORD=ваш_пароль
DB_PREFIX=viz620
```

### Локальный запуск без Docker

1. Установите PHP 5.6+ и MySQL 5.7
2. Настройте веб-сервер (Apache с mod_rewrite)
3. Импортируйте БД из `database/structure.sql`
4. Отредактируйте `admin/_mysql.php`

## 🔐 Безопасность

### Для продакшена:

1. **Смените пароли** в `.env`
2. **Включите HTTPS**
3. **Настройте бэкапы**
4. **Обновите PHP** до актуальной версии (требуется рефакторинг CMS)

### Известные проблемы:

- ⚠️ Используется устаревший `mysql_*` API (PHP 5.x)
- ⚠️ Нет CSRF-защиты
- ⚠️ Пароли админов хранятся как MD5
